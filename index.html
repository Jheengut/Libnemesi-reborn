Libnemesi-reborn by syhou

An RTSP/RTP/RTCP library implemented by C. Maintenance job is doing by Syhou. Contact me: reckhou@me.com

# README # Syhou: reckhou@me.com You can use LibNeMesi to dump real time stream (like H264 or AAC ...) to files. The stream is mainly from RTSP server.

Install

1. Configuration When first build, you need configure libnemesi and netembryo. Just do as below: $ cd netembryo-0.0.5 $ ./configure $ cd ../libnemesi-0.6.4-rc2.modify $ ./config.sh 2. Making and Running $ make $ make clean
